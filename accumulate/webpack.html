# webpack的入门学习

webpack对应前端而言是一个打包工具等等（反正就很多）

# 初始化项目

  • 建目录，初始化
npm init
  • 安装webpac
npm i -D webpack webpack-cli
  • new 一个文件夹src，然后再new一个文件main.js,测试打包是否能成功
console.log('hello word')
  • 配置webpack.json的打包命令
 "scripts":{
     "build": "webpack src/main.js"
 }
  • 运行
 npm run build

：

完成后会有一个dist的文件夹，打开它
